*****************
Fortran Interface
*****************

The Fortran interface comes in two flavors. The high-level interface
is suitable for most end-user applications, while the low-level
interface requires more work to use but offers greater
flexibility. The low-level interface also provides the core
functionality that the :ref:`python-interface` and :ref:`c-interface`
are built upon.
